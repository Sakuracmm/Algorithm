动态规划第三节 - 序列型动态规划

- 有状态的序列型动态规划

- 特点
    - 题目变种多
    - 没有固定的模板
    - 见招拆招

- 给定一个序列
- 动态规划方程f[i]中的下标i表示【前i个元素a[0],a[1], ..., a[i - 1]的某种性质】
    - 坐标型的f[i]表示以ai结尾的某种性质
-因为初始化的时候，f[0]可以表示空序列的性质
    - 坐标型动态规划的初始条件f[0]就是指以a0结尾的子序列的性质


-小结
    当思考动态规划最后一步时，这一步的选择依赖前一步的某种状态
    初始化的时候，dp[0]代表前0个元素/前0天的情况
        -与坐标型动态规划的区别

    计算的时候，f[i]代表前i个元素(即 0 ~ i - 1)的某种性质

    -最长序列型动态规划（其实是坐标型动态规划）
    -要求找出符合条件的最长子序列
    -方法
        -记录以每个元素i结尾的最长子序列的长度
        -计算时，在i之前枚举子序列上一个元素是哪个
     LongestIncreasingSubsequence - 76
