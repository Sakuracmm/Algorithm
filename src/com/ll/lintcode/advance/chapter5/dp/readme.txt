1. 动态规划四要素：
    - 灵感，创造力，存储小规模问题的结果
        - 最优解/Maximum/Minimum
        - Yes/No
        - Count(*)
    - 方程 Function
        - 状态之间的联系，怎么通过小的状态，来求得大的状态
    - 初始化 Initialization
        - 最极限的小状态是什么，起点
    - 答案 Answer
        - 最大的那个状态是什么，终点

2. 滚动数组的优化
    f[i] = max(f[i - 1], f[i - 2] + A[i]);
    --->>>
    f[i % 2] = max(f[(i - 1) % 2] 和 f[(i - 2) % 2])

3. 小技巧
    网格类的题目正方形用右下角作为定位角
    长方形可以用左上角和右下角作为定位角
